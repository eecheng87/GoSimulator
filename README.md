# GoSimulator

## How to build

change directory:
```
> cd GoSimulator
```

install dependencies:
```
> npm install
```

run the app:
```
> SET DEBUG=gosimulator:* & npm start
```

open web through url: [http://localhost:3000/](http://localhost:3000/)

## Develop mode
open server through `nodemon`, it will re-run when file was changed
```
> nodemon ./bin/www
```
